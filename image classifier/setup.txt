===== Instructions to setup ====

1. Import requirements

To do this step, open the terminal.
Once the terminal is open, type the following command:
cd 'image classifier'
Now that you are in the image classifier directory, type the following command:
pip install -r requirements.txt

2. Begin training

To begin training, open the app.py file.
Once the file is open, depending on the IDE being used, you can press the play button.
If you don't see a play button, open the terminal and cd into the 'image classifier' directory again.
Now type the following command:
py app.py
